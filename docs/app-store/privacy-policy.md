## 一、总则

Engram(下称"本 App")是一款帮助创作者理解与沉淀短视频内容的工具。我们把"隐私优先、数据留在你设备上"作为基本原则:**核心分析在你的设备本机完成,我们不运营任何服务器,也不收集、不上传你的个人信息到我们这里。**

## 二、我们如何处理你的信息

### 1. 完全在本机处理(默认)
以下都在你的设备上离线完成,不离开设备:
- 你从相册选择/分享导入的**视频文件**;
- 语音转文字(设备端语音识别)、画面文字识别(设备端 OCR)、关键帧抽取;
- 内容的检索索引、问答、拆解与洞察;
- 所有拆解结果、剪藏、剧本、设置等,均**保存在本 App 的本地存储**中。

### 2. 可选的云端 AI(由你自行配置)
当且仅当你在"设置"中主动配置并选择相应模式时,本 App 才会调用你选择的第三方服务:

- `Ark Standard`:向用户配置的 Ark/OpenAI-compatible 服务发送抽样代表帧与文本,不上传完整视频;
- `LAS Deep`:使用用户提供的 LAS API Key 与短期 TOS STS 凭据,把用户明确选择的一份视频以流式 multipart 方式临时上传至用户自己的 TOS 前缀,再让 LAS 视频分镜、视频精细理解、短剧/电影剧本生成和增强 ASR 复用该对象;
- `LAS + Ark Refine`:先完成 LAS Deep,再仅把低置信镜头的代表帧发送给 Ark。

LAS 的真实能力探测会使用用户主动选择的非私密小视频、LAS API Key 和临时 TOS STS,并可能产生第三方费用。正式分析还要求针对实际视频、run 和精确 plan 的一次性上传/费用同意;费用无法核实时显示"未知/可能收费"。缺少或过期的任一角色会阻断执行并显示精确原因,只允许用户显式选择 Ark Standard、Local 或取消,不会静默上传或降级。

关于这一点请知悉:
- 这些数据**只发送到你选择的第三方服务**(例如火山方舟/豆包等),**不经过、也不发送给本 App 开发者**——我们没有服务器;
- 该第三方如何处理这些数据,**受其自身的服务条款与隐私政策约束**,请你自行阅读并同意;
- Ark Key、LAS Key 与 TOS 临时 AccessKey/Secret/SecurityToken 保存在彼此独立的系统钥匙串(Keychain)项中,仅用于向你配置的服务发起请求,绝不发送给开发者、绝不明文存储在普通文件里;当前路径不接受长期 TOS AK/SK。
- 云端错误与诊断在保存前会移除密钥、令牌和本地文件路径;本地分析产物与导出文件仍由你自行控制和删除。
- 原视频任务完成、取消或失败时会立即尽力删除临时 TOS 对象;失败会显示 cleanup-pending 并保留最长 24 小时的重试状态。LAS 生成的分镜/剧本结果可保留在用户自己的 TOS 与本地文档中,由用户自行查看和删除。预签名下载 URL、原始服务商响应和凭据不会写入普通持久化、日志或导出。
- 个人临时 STS BYOK 不等于公开发布方案。公开启用完整视频上传前仍需真实的 presigned broker 后端、部署责任人及签发/删除验证;在该边界完成前 LAS 公开发布保持阻断。

### 3. 我们不做的事
- 不设置任何分析/统计/广告 SDK,**不追踪**你;
- **没有开发者后端**,不收集你的账号、位置、通讯录、设备标识等个人信息;
- 不会在你未配置云端服务的情况下,把任何内容上传到网络。

## 三、权限说明
- **语音识别**:用于把你导入视频里的语音转成文字,**在本机进行**;
- **相册**:通过系统图片选择器(PHPicker)选取你要导入的视频,本 App **不读取你的相册整体**,仅接收你主动选择的那个文件。

## 四、数据存储与删除
- 数据存储在本 App 的本地容器中;你可在 App 内**删除**任意剪藏/拆解,删除会移除对应的本地数据与索引;
- 如果你选择 LAS,临时原视频和生成结果还会按上一节说明出现在你自己的 TOS 中;原视频由 App 立即清理/重试,生成结果由你按自己的 TOS 保留策略控制;
- 卸载本 App 会移除其本地数据。

## 五、第三方与共享
- 除"你主动配置的云端 AI 服务"外,本 App **不与任何第三方共享**你的信息;
- 本地模型权重从相应提供方(如 Hugging Face)按需下载,下载行为遵循其条款。

## 六、儿童
本 App 不面向 14 岁以下儿童,也不有意收集其个人信息。

你对设备上的数据拥有完全控制权,可随时查看、导出(分享)、删除。如有疑问,请通过上方联系方式与我们联系。
